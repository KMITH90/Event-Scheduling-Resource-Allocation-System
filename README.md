# Event Scheduling & Resource Allocation System

An intelligent web-based application designed to simplify event scheduling and optimize resource allocation for organizations, colleges, and businesses. The system helps users efficiently manage events, venues, staff, and resources while avoiding scheduling conflicts and improving overall planning.

---


## Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python / Flask
- **Database:** SQLite / MySQL
- **Version Control:** Git & GitHub

---

### Install Dependencies

```bash
pip install -r requirements.txt
```

If `requirements.txt` is unavailable, install Flask manually:

```bash
pip install flask
```

---

## Run the Application

```bash
python app.py
```

Open your browser and visit:

```bash
http://127.0.0.1:5000/
```
